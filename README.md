## 💡 О проекте

**AI Helper Budz** — это современное SPA-приложение на связке Laravel + Vue.js, в котором пользователь может:

- 🔐 зарегистрироваться и войти в систему
- 💬 общаться с AI через чат (поддержка markdown, файлов)
- 📋 просматривать историю диалогов
- 💳 оформить подписку на один из тарифов
- 👤 управлять профилем и настройками аккаунта
- 📊 работать с лимитами запросов

Проект разработан как **курсовая работа** с упором на современную архитектуру, чистый код и приближение к production-подходу.

## ⚙️ Функциональность

### 🔐 Авторизация
- Регистрация нового пользователя
- Вход и выход из аккаунта
- Защита всех API-маршрутов через Sanctum

### 💬 AI-чат
- Создание новых чатов
- Отправка и получение сообщений от AI
- Поддержка синтаксиса Markdown в ответах
- История диалогов в боковой панели
- Загрузка файлов в чат

### 💳 Подписки
- Страница выбора тарифного плана
- Разные описания и условия для каждого тарифа
- Многошаговый UI: выбор → оплата → обработка → успех
- Валидация данных карты на клиенте и на сервере
- Имитация процесса оплаты с прогресс-баром
- Возможность сохранить карту (данные шифруются AES-256)
- Отображение активной подписки в профиле

### 👤 Профиль пользователя
- Просмотр статуса и срока подписки
- Смена email и пароля
- Удаление аккаунта

### 📊 Лимиты запросов
- 12 бесплатных запросов для новых пользователей
- Безлимитный доступ при активной подписке

## 🚀 Установка и запуск

### Требования
- PHP 8.2+
- Composer
- Node.js 20+
- PostgreSQL 14+

### 1. Клонирование
```bash
git clone https://github.com/Amudz1/kursach_v2.0.git
cd kursach_v2.0
```

### 2. Зависимости
```bash
composer install
npm install
```

### 3. Настройка `.env`
```bash
cp .env .env.backup
```

Откройте `.env` и заполните обязательные поля:
```env
APP_NAME="AI Helper Budz"
APP_ENV=local
APP_DEBUG=true
APP_URL=http://localhost:8000

DB_CONNECTION=pgsql
DB_HOST=127.0.0.1
DB_PORT=5432
DB_DATABASE=your_database
DB_USERNAME=your_username
DB_PASSWORD=your_password

OPENROUTER_API_KEY=sk-or-v1-ВАШ_КЛЮЧ   # https://openrouter.ai/keys
```

### 4. Инициализация
```bash
# Генерация ключа
php artisan key:generate

# Миграции + тестовые данные
php artisan migrate:fresh --seed

# Ссылка на хранилище
php artisan storage:link
```

### 5. Запуск

**Терминал 1 — PHP сервер:**
```bash
php artisan serve
```

**Терминал 2 — Vue сборка:**
```bash
npm run dev
```

Откройте браузер: **http://localhost:8000**
